# Infinity Nikki Toolkit (INToolkit) 
一个专为无限暖暖而生的工具箱，采用 MIT 许可证开源。旨在提升桌面端玩家的游戏体验。
使用有关的官方资源与第三方资源通过开发团队的设计结合为高效易用的工具箱。

**请注意：它并不对游戏客户端进行任何破坏性修改，请在使用前务必知悉[相关许可协议](#有关许可协议临时)之事项。因使用本工具造成任何意外损失问题。作者均不承担任何后果**

## 技术栈
- .NET 8.0
- WinUI 3
